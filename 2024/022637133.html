毛刺过滤电路

同时过滤正毛刺和负毛刺

用verilog设计一个功能模块，将不足两个时钟周期长度的毛刺过滤掉，时序图如下所示：

输入信号：clk，rst_n，Din

输出信号：Dout

思路：如果两个周期输入信号保持不变，则输出。否则，保持不变；

module filter(
    input       17clk,
    input       nrst,
    input       din,
    output reg  dout 
); 
reg din_r; 
always @(posedge clk or negedge nrst) begin
    if(!nrst) begin
        din_r  <= 0;
    end else begin
        din_r  <= din;
    end
end
always @(posedge clk or negedge nrst) begin
    if(!nrst) 
        dout <= 0;
    else if(din == din_r)
        dout <= din;
end
endmodule

仅过滤单向毛刺

  • 过滤负毛刺

    过滤一个周期的负毛刺，当负电平持续两个周期，才可以输出低电平；

    采用两拍的寄存器，两级输出做 “或”操作；

  • 过滤正毛刺

    过滤一个周期的正毛刺，当正电平持续两个周期，才可以输出正电平；

    采用两拍的寄存器，两级输出做 “与”操作；

    module filter_pos(
    input       clk,
    input       nrst,
    input       din,

    output      dout
);
always @(posedge clk or negedge nrst) begin
    if(!nrst)
        din_r <= 0;
    else
        din_r <= din;
end
assign dout = din & din_r;

消抖动电路

题目：用verilog实现按键抖动消除电路，抖动小于15ms，输入时钟12MHz。

思路：添加计数器，如果15ms数据没有变化，将输入放在输出。

module de_jitter (
    input       clk,
    input       nrst,
    input       din,
    
    output      dout
);
localparam jitter_factor = 180000;

reg [17:0] cnt;
reg din_r;

always @(posedge clk or negedge nrst) begin
    if(!nrst)
        din_r <= 0;
    else 
        din_r <= din;
end

always @(posedge clk or negedge nrst) begin
    if(!nrst) 
        cnt <= 'h0;
    else if(!flag)
        cnt <= 'h0;
    else 
        cnt <= cnt + 'h1;
end

always @(posedge clk or negedge nrst) begin
    if(!nrst) 
        flag <= 1;
    else if(din_r == din)
        flag <= 1;
    else 
        flag <= 0;
end

always @(posedge clk or negedge nrst) begin
    if(!nrst) 
        dout <= 0;
    else if(cnt == jitter_factor - 1)
        dout <= din;
end
endmodule

串并、并串转换

串转并

​ 用一个计数器count，每输入8个数，就输出一次，每周期dout_temp左移一位，然后再将输入的1bit串行据存入dout_temp的最低位。

module Serial_Parallel(
    input            clk,
    input            nrst,
    input            data_in,
    input            data_valid,

    output reg [7:0] data_out,
    output           data_out_valid
);

reg [2:0] cnt; 
reg [7:0] data_reg;
always @(posedge clk or negedge nrst) begin
    if(!nrst)
        cnt <= 0;
    else if(data_valid) begin
        if(cnt == 7)
            cnt <= 0;
        else
            cnt <= cnt + 'h1;
    end
end
always @(posedge clk or negedge nrst) begin
    if(!nrst)
        data_reg <= 'h0;
    else if(data_valid)
        data_reg <= {data_reg[6:0], data_in};
end
always @(posedge clk) begin
    if(cnt == 7) begin
        data_valid <= 1;
        data_out <= data_reg;
    end else begin
        data_valid <= 0;
        data_out <= data_reg;
    end
end
endmodule

并转串

module Parallel_Serial(
    input           clk,
    input           nrst,
    input [7:0]     data_in,
    input           data_in_valid,

    output          data_out_valid, 
    output          data_out
);
reg [3:0] cnt;
reg [7:0] data_in_reg;
always @(posedge clk or negedge nrst) begin
    if(!nrst) 
        data_in_reg <= 'h0;
    else if(data_in_valid) 
        data_in_reg <= data_in;
    else 
        data_in_reg <= data_in_reg << 1;
end
always @(posedge clk or negedge nrst) begin
    if(!nrst)
        cnt <= 0;
    else if(data_in_valid)
        cnt <= 8;
    else if(cnt >= 1)
        cnt <= cnt - 1;
end
assign data_out_valid = (cnt >= 1)?1:0;
assign data_out = data_in_reg[7];
endmodule

数据位宽转换

宽到窄传输

​ 假设数据从模块A传入到模块B，模块A的输出数据为32位，模块B的输入数据位宽为16位，那么如何能把数据从A传入B而不损失数据呢。

思路：通过时钟分频和倍频

宽数据是慢时钟，窄数据是快时钟；

Input：快时钟+宽数据

Output：窄数据（快时钟输出）

module Wide_Narror(
    input           clk_2x,
    input           nrst,
    input [63:0]    data_in,
    
    output [31:0]   data_out
);

reg clk_1x;
reg flag;

always @(posedge clk_2x or negedge nrst) begin
    if(!nrst)
        clk_1x <= 0;
    else 
        clk_1x <= ~clk_1x;
end

always @(posedge clk_1x or negedge nrst) begin
    if(!nrst)
        data_in_reg <= 'h0;
    else 
        data_in_reg <= data_in;
end

assign data_out = clk_1x ? data_in_reg[63:32] : data_in_reg[31:0];

endmodule

窄到宽传输

宽数据是慢时钟，窄数据是快时钟；

Input：快时钟 + 窄数据

Output：宽数据（慢时钟输出）

module Narror_Wide(
    input           clk_2x,
    input           nrst,
    input [31:0]    data_in,
    
    output [63:0]   data_out
);

reg clk_1x;
reg flag;

always @(posedge clk_2x or negedge nrst) begin
    if(!nrst)
        clk_1x <= 0;
    else 
        clk_1x <= ~clk_1x;
end

always @(posedge clk_1x or negedge nrst) begin
    if(!nrst)
        data_in_reg <= 'h0;
    else 
        data_in_reg <= {data_in_reg[31:0], data_in};
end

assign data_out = clk_1x ? data_in_reg : 'h0;

endmodule

握手解决流水线断流、反压问题

通过三级流水线实现以下组合逻辑的计算；包含握手和反压的处理。

y = (c1+c2)*(c3+c4) + a2*b2 + a3*b3 + a4*b4

思路：第一级流水完成 (c1+c2)，(c3+c4)的计算
第二级流水完成乘法运算
第三级流水加法运算

每一级的上下游都要有握手和反压信号的处理；

  • ready信号的处理

    以ready_r1为例，当r1寄存器为空（valid_r1==0）或者ready_2为1时，r1都有能力接收数据；
    如果ready_r2为1，valid_r1也为1，则将该结果保存在r2寄存器中；下一个周期r1可以接收新数据；
    如果ready_r2为0，valid_r1为1，下级无法握手成功传输，所以r1无法接收新数据；

  • valid信号的处理

    对于r1来说，valid_r1可以作为握手的结果信号；当valid_i和ready_r1握手成功后，r1输出valid_r1为1和有效的data_a。

module handshake_pipline(
    input       clk,
    input       nrst,

    input       valid_i,       
    input [7:0] c1,
    input [7:0] c2,
    input [7:0] c3,
    input [7:0] c4,
    input [7:0] a2,
    input [7:0] b2,
    input [7:0] a3,
    input [7:0] b3,
    input [7:0] a4,
    input [7:0] b4

    output reg [15:0] result,
    output      valid_o,

    input       ready_i,
    output      ready_o
);

reg [7:0] a1_r, b1_r;
wire ready_r1, ready_r2, ready_r3;

// first stage
assign ready_r1 = ready_r2 | (!valid_r1);

always @(posedge clk or negedge nrst) begin
    if(!nrst)
        valid_r1 <= 0;
    else if(ready_r1)
        valid_r1 <= valid_i;
end

always @(posedge clk) begin
    if(valid_i & ready_r1) begin
        a1_r <= c1 + c2;
        b1_r <= c3 + c4;
    end
end

reg [7:0] a2_r, b2_r, a3_r, b3_r, a4_r, b4_r;
always @(posedge clk) begin
    if(valid_i & ready_r1) begin
        a2_r <= a2;
        b2_r <= b2;
        a3_r <= a3;
        b3_r <= b3;
        a4_r <= a4;
        b4_r <= b4;
    end
end

// second stage
assign ready_r2 = ready_r3 | (!valid_r2);
always @(posedge clk or negedge nrst) begin
    if(!nrst)
        valid_r2 <= 0;
    else if(ready_r2)
        valid_r2 <= valid_r1;
end

reg [15:0] mul_1, mul_2, mul_3, mul_4;
always @(posedge clk) begin
    if(valid_r1 & ready_r2) begin 
        mul_1 <= a1_r * b1_r;
        mul_2 <= a2_r * b2_r;
        mul_3 <= a3_r * b3_r; 
        mul_4 <= a4_r * b4_r;
    end
end

// third stage
assign ready_r3 = ready_i | (!valid_r3);
always @(posedge clk or negedge nrst) begin
    if(!nrst)
        valid_r3 <= 0;
    else if(ready_r3)
        valid_r3 <= valid_r2;
end

always @(posedge clk) begin
    if(valid_r2 & ready_r3) begin
        result <= mul_1 + mul_2 + mul_3 +mul_4;
    end
end

assign valid_o = valid_r3;

endmodule

快速计算one-hot码中“1”所对应的下标

题目：找一个多bit信号最高位的1或最低位的1所对应的下标

思路：根据独热码和对应坐标的真值表，写出逻辑表达式；

module One_Hot_position(
    input       clk,
    input       nrst,
    input [7:0] data_in,
    output[2:0] position
);
assign position[0] = data_in[1] | data_in[3] | data_in[5] | data_in[7];
assign position[1] = data_in[2] | data_in[3] | data_in[6] | data_in[7];
assign position[2] = data_in[4] | data_in[5] | data_in[6] | data_in[7];
//    one_hot	index
// 8'b0000_0001	3'b000
// 8'b0000_0010	3'b001
// 8'b0000_0100	3'b010
// 8'b0000_1000	3'b011
// 8'b0001_0000	3'b100
// 8'b0010_0000	3'b101
// 8'b0100_0000	3'b110
// 8'b1000_0000	3'b111
endmodule

找到序列第一个1和最后一个1

思路：找出多bit数据中第一个1和最后一个1的位置，将其转换成独热码，根据独热码得到位置；

通过”按位”操作

// MAX
pre[7] = 0;
pre[6] = pre[7] | data_in[7];
pre[5] = pre[6] | data_in[6];
......
~pre & data_in => One_Hot
// MIN
// 利用补码的特性，原码和补码”与“操作后，得到独热码，1的位置为原码最低位为1的位置
data_in & ~(data_in - 1) => One_Hot
module Find_Max_1(
    input       clk,
    input       nrst,

    input [7:0] data_in,
    output[7:0] data_out
);
wire [7:0] pre;

assign pre[7] = 0;
assign pre[6:0] = pre[7:1] | data_in[7:1]; 

assign data_out = ~pre & data_in;
endmodule
module Find_Min_1(
    input       clk,
    input       nrst,

    input [7:0] data_in,
    output[7:0] data_out
);

assign data_out = data_in & ~(data_in - 1);

endmodule

跨时钟域传输

已知data a为a时钟域0-15范围内的顺序数，现需要将它的值传递至b时钟域(设b的时钟频率高于a)，请便用verilog设计合适的跨时钟域处理逻辑、模块端口定义如下:

module data_sync(
  input         nrst,
  input [3:0]   data_a,
  input         aclk,

  output        bclk,
  output [3:0]  data_b
);

由于计数值是连续变化的，采用格雷码实现。

module data_sync(
  input         nrst,
  input [3:0]   data_a,
  input         aclk,

  output        bclk,
  output [3:0]  data_b
);

wire [3:0] data_a_gray; 
reg  [3:0] data_a_gray_r; 

assign data_a_gray = data_a ^ (data_a >> 1);
always @(posedge aclk or negedge nrst) begin
  if(!nrst)
    data_a_gray_r <= 'h0;
  else
    data_a_gray_r <= data_a_gray;
end

always @(posedge bclk or negedge nrst) begin
  if(!nrst) begin
    data_b_gray <= 'h0;
    data_b_gray_r <= 'h0;
  end else begin
    data_b_gray <= data_a_gray;
    data_b_gray_r <= data_b_gray;
  end
end

assign data_b[3] = data_b_gray_r[3];
assign data_b[2] = data_b_gray_r[2] ^ data_b[3];
assign data_b[1] = data_b_gray_r[1] ^ data_b[2];
assign data_b[0] = data_b_gray_r[0] ^ data_b[1];

endmodule

模3 模7检测器

使用verilog代码，设计电路，判断输入序列能否被三整除，能的时候输出1，不能的时候输出0

思路：本质是一个序列检测器，通过现态和输入决定输出和下一个状态。余数只有三种可能，00,01,10。输入从右面进，之前的状态需要向左移1比特。状态机如下。

当前态余数 输入 次态余数 输出
0 1 1 0
0 0 0 1
1 0 2 0
1 1 0 1
2 0 1 0
2 1 2 0 
module mod3(
  input       clk,
  input       nrst,
  input       data_in,

  output reg  data_out
);
parameter [1:0] S0 = 2'b00,
                S1 = 2'b01,
                S2 = 2'b10;
reg [1:0] state, next_state;

always @(posedge clk or negedge nrst) begin
  if(!nrst) begin
      state <= S0;
  end
      state <= next_state;
end

always @(*) begin
  case(state) begin
    S0: if(data_in) next_state = S1; else next_state = S0;
    S1: if(data_in) next_state = S0; else next_state = S2;
    S2: if(data_in) next_state = S2; else next_state = S1;
    default: next_state = S0;
  end
end

always @(*) begin
  if(((state == S0) && (!data_in)) || ((state == S1) && (data_in)))
    data_out = 1;
  else 
    data_out = 0;
end

endmodule
